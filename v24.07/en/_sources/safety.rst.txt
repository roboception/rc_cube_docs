Intended use

The

rc_cube

 is intended to be used in combination with a 3D camera for data
acquisition (e.g., stereo images). It is furthermore intended to
process that data using 3D-image processing algorithms to serve in
applications such as object detection or robotic pick-and-place.

Warning: The

  rc_cube

   is only intended for stationary installation.

Warning: The

  rc_cube

   is

  NOT

   intended for safety critical applications.
